Quicken User's Guide

Not all accounting systems use the same words for the same concepts. Below follws some notes for users accustomed to Quicken (TM) products.

Categories

What Quicken calls "Categories" are really just Income/Expense accounts. Thus, if you are used to specifying a category in Quicken, just create an income/expense account of the same name in X-Acccountant, and use that as a category.
