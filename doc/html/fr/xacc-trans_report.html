Rapport de transaction

Les rapports de transaction liste en détails les transactions dans un compte spécifié sur une période spécifiée, dans un  format convenu pour la visualisation et l'exportation.

Les options à spécifier pour ce rapport sont:

  • Compte: le compte sur lequel l'on fait le rapport.
  • La Date jusqu'à et la Date depuis: l'échelle de date que le rapport couvre.
  • Clé de tri primaire : le champ à trier en premier.
  • Ordre de tri primaire: l'ordre dans lequel il faut trier. "Ascendant" specifie du plus petit vers le plus grand, par tri alphabétique, du plus ancien vers le plus récent, ou les plus petits montants en premier. Le Descendant produit l'effet opposé/contraire.
  • Clé de tri secondaire: le champ à trier s'il y a des clés de tri primaire équivalentes/égales
  • Ordre de tri secondaire: l'ordre dans lequel se fait  le tri de la clé secondaire.
